
# Complex React App

Social media app, with posting, chats and token authentication.

## Usage/Examples

- npm install
- npm run dev

Go to http://localhost:3000/


### Tech Stack
- React 
- React-Router
- React Context Api
- Immer
- Axios
- Socket.IO
- Webpack
- Html
- CSS
